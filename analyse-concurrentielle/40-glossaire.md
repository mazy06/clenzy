# Glossaire — les mots à connaître

> Tous les termes du métier et du produit, en une ligne chacun. À garder sous la main en lisant le reste du document.

## Le métier de la location courte durée

| Terme | En clair |
|---|---|
| **PMS** (Property Management System) | Le logiciel qui gère toute la location courte durée : réservations, calendrier, ménage, paiements, communication. **C'est ce qu'est Baitly.** |
| **Location courte durée / STR** | *Short-Term Rental* : louer des logements à la nuitée (l'univers Airbnb). |
| **OTA** (Online Travel Agency) | Les plateformes de réservation : Airbnb, Booking, Vrbo… Elles prennent une **commission** (15-20 %). |
| **Conciergerie** | Une société qui gère des logements Airbnb **pour le compte de propriétaires**. **Notre cible cœur.** |
| **Gestionnaire / host** | Celui qui exploite les logements (le client de Baitly). |
| **Channel manager** | Un outil qui connecte un logement à **plusieurs OTA en même temps** et synchronise les calendriers. |
| **Booking direct / site direct** | Réserver via le site du gestionnaire, **sans OTA** → pas de commission, plus de marge. |
| **Reversement (payout)** | L'argent qu'on **reverse au propriétaire** une fois le séjour passé. |
| **Caution / pré-autorisation** | Bloquer une somme sur la carte du voyageur **sans la débiter** (garantie). |
| **Surbooking (overbooking)** | Deux réservations sur les mêmes dates — l'erreur à éviter absolument. |

## Le produit Baitly

| Terme | En clair |
|---|---|
| **La Constellation** | Notre **équipe d'agents IA** (Finance, Opérations, Revenue, Communication, Réputation) qui **proposent et exécutent** des actions. |
| **HITL** (*Human In The Loop*) | « Humain dans la boucle » : **l'IA propose, un humain valide** avant l'exécution. |
| **Autonomie déterministe** | Quand une action est validée assez souvent, l'IA finit par l'exécuter **seule**, selon des règles claires. |
| **Moteur Ménage** | Notre brique qui **calcule le prix d'un ménage**, paie le prestataire **à la preuve photo**, et transforme les anomalies en devis. |
| **Moat / fossé** | Un **avantage difficile à copier**. Baitly en a deux : conformité multi-pays + économie du ménage. |
| **Multi-tenant** | Une seule application qui héberge **plusieurs organisations**, chacune isolée des autres. |
| **Yield / tarification dynamique** | Ajuster les prix automatiquement selon la demande, la saison, les événements. |

## Conformité & finance

| Terme | En clair |
|---|---|
| **Factur-X / e-invoicing** | La **facture électronique**, bientôt **obligatoire** en France (2026). |
| **PDP** | *Plateforme de Dématérialisation Partenaire* : un opérateur **agréé** pour transmettre les factures électroniques. |
| **NF525** | La **norme française anti-fraude** pour les logiciels de facturation. |
| **ZATCA** | L'administration fiscale **saoudienne** (facture électronique en Arabie Saoudite). |
| **KYC** (*Know Your Customer*) | La **vérification d'identité**, obligatoire pour manipuler des paiements. |
| **Déclaration voyageurs** | L'obligation légale de **déclarer les voyageurs** (fiche de police en France, Shomoos/DGSN en Arabie Saoudite). |
| **Trust accounting** | La gestion **séparée et traçable** de l'argent des propriétaires (comme un compte séquestre). |

## Divers

| Terme | En clair |
|---|---|
| **ARI** | *Availability, Rates, Inventory* : les 3 infos qu'on synchronise avec les OTA (**dispo, prix, stock**). |
| **RICE** | Une méthode pour **prioriser** : Portée × Impact × Confiance ÷ Effort. |
| **MENA** | *Middle East & North Africa* — notre zone d'**expansion** (Maroc, Arabie Saoudite). |
| **SWOT** | Forces, Faiblesses, Opportunités, Menaces. |
| **GTM** (*Go-To-Market*) | La stratégie pour **aller chercher les premiers clients**. |
| **ARPA** | *Average Revenue Per Account* : le **revenu moyen par client**. |
